## まとめ

- PyCharm で Invalid Python interpreter selected for the project が出た際には設定の Interpreter 一覧から Invalid なものを全部削除すると直るかも
- PyCharm 管理配下の venv をターミナルから削除した際に発生したエラーで、過去の venv まわりの記録が 微妙に残ってしまっているのが原因と思われる
