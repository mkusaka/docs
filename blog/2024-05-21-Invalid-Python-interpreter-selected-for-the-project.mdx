## まとめ

* PyCharm で Invalid Python interpreter selected for the project が出た際には .idea フォルダを一旦消して再起動すれば直るかも
* PyCharm 管理配下の venv をターミナルから削除した際に発生したエラーで、過去の venv まわりの記録が .idea フォルダ配下に残ってしまっている様子
