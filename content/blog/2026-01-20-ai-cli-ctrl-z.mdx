## まとめ

- Claude Code / OpenCode / Codex CLI はいずれも Ctrl+Z でサスペンドして `fg` で復帰できる（Unixのジョブコントロール）
- vimでCtrl+Z → シェル作業 → fgで戻る、と同じ要領
- セッションやコンテキストは維持される
- `ls` や `git status` など、ちょっとしたコマンドでディレクトリの状況確認したいときに重宝する

## 使い方

```bash
# AI CLIで作業中に Ctrl+Z
[1]+  Stopped                 claude

# シェルで好きなコマンドを実行
$ ls
$ git status

# fg で復帰
$ fg
```
